enum ssd1608_hink_part_no

HINK Part Selection enum.

HINK Display part number - SSD1608 supports both the HINK-E0154A05 and the HINK-E0154A45. The library needs to know which so it can adjust the initialization accordingly.

◆ ssd1608_lut_selection

enum ssd1608_lut_selection

LUT Selection enum.

Look Up Table for waveforms are stored in a OTP on the screen, but can also be loaded into a register on the display driver. This allows for custom waveforms to be used to alter the display refresh pattern.
