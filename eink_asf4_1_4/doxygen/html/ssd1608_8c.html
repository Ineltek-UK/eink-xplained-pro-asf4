void ssd1608_init ( void  )

Initializes the SSD1608 Eink display driver.

Call this function to initialize the hardware interface and the SSD1608 display driver. When initialization is done the display is ready to receive data.

Set the SSD1608 configuration registers.

Sets all the necessary configuration for the SSD1608 Eink display driver to communicate with the MCU and sets the necessary variables from the config struct to allow the display to work.
