void eink_read_data ( uint8_t  command,
uint8_t *  data 
)

Writes a command and read returned data byte from the Eink display driver.

Sends data over the SPI interface, initially with a command followed by a read of the returned data. Ensures the SPI interface is correctly set before sending raw data. Switches the DATA line to an input ready for receive operation. Returns the DATA line to an output once complete.

Parameters
commandCommand byte to write.
*dataPointer to data byte to read into.

void eink_send_raw_byte ( uint8_t  send_byte,
uint8_t  f_send 
)

Writes raw data to the Eink display driver.

Writes raw SPI data bits to the selected Eink display driver along
with a clock. Also sets CSB pin to enable data transmission. Note that no other SPI pins are changed. For correct SPI transmission, use eink_write_data() or eink_read_data().

Parameters
send_byteData byte to write.
